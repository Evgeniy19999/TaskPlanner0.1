Планировщик задач на С++(версия с ООП)

Консольное приложение для управления задачами.
Поддерживает:
-Добавление задач
-Просмотр всех задач
-Редактирование и удаление
-Сохранение/загрузка из файла 'tasks.txt'

Использованные технологии:
-Язык С++
-ООП: классы, инкапсуляция

Файлы проекта.
-file_utlis.cpp
-file_utlis.h
-main.cpp
-task.cpp
-task.h
